# 飞书股票智能机器人 - 功能清单

## 🎯 已实现的功能

### ✅ 核心功能

1. **双向问答系统**
   - ✅ 飞书接收消息 → WebSocket服务器
   - ✅ 转发到Coze智能体 → 获取回复
   - ✅ 推送回复到飞书（包含问题和回答）

2. **定时自动推送**
   - ✅ 每天 09:00、15:00、20:00 自动推送
   - ✅ 支持多用户订阅/取消订阅
   - ✅ 可自定义推送时间和问题

3. **上下文连贯对话**
   - ✅ 维持对话历史（最近20条）
   - ✅ 每个用户独立上下文
   - ✅ 支持清空上下文

4. **图片分析功能**
   - ✅ 接收飞书图片消息
   - ✅ 调用豆包API分析图片
   - ✅ 将分析结果转发给Coze智能体
   - ✅ 推送完整分析结果

5. **本地WebSocket长连接**
   - ✅ 无需公网服务器
   - ✅ Windows后台挂起运行
   - ✅ 自动重连机制

### ✅ 命令功能

| 命令 | 功能 | 状态 |
|------|------|------|
| `订阅` | 订阅股票分析推送 | ✅ |
| `取消订阅` | 取消订阅 | ✅ |
| `立即分析` | 立即获取股票分析 | ✅ |
| `清空上下文` | 清空对话历史 | ✅ |
| `帮助` | 显示帮助信息 | ✅ |
| `自由对话` | 直接与智能体对话 | ✅ |
| `发送图片` | 分析图片并获取相关建议 | ✅ |

```

```

## 📞 测试方法

1. **启动机器人**
   ```bash
   launch.bat
   ```

2. **在飞书中测试**
   - 发送 `帮助` → 查看可用命令
   - 发送 `订阅` → 订阅股票分析
   - 发送 `立即分析` → 立即获取分析
   - 发送 `清空上下文` → 清空对话
   - 发送任意问题 → 与智能体对话
   - 发送图片 → 分析图片内容

3. **查看日志**
   ```bash
   Get-Content feishu_bot.log -Wait
   ```
